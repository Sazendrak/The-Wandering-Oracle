**Dhampir** (Errata)

[LEGACY GATE DISCLAIMER]

This errata applies to a standard 2014 spell/feature that does not have a corresponding page in the TWR Wiki. It is preserved here for completeness.

The dhampirs empowered bite attack only grants a bonus to the ability check granted by the bite, without the addition of other modifiers.
